Cristopher De La O

Originalmente, este proyecto no estaba planeado, ya que utilizaba una hoja de Excel para hacer el análisis y generar mis propios horarios, sin embargo, la baja escalabilidad me llevó a tener que modificar fórmulas y mover datos constantemente.

Debido a esta situación, decidí implementar una aplicación para realizar el procedimiento de forma sistematizada. Personalmente, me pareció una gran idea, por lo que me decante por tecnologías multiplataforma, con el fin de que cualquiera en la comunidad pueda utilizar la aplicación sin necesidad de instalaciones ni compilaciones de código.

Se aceptan sugerencias para la mejora de la aplicación, las peticiones se pueden realizar a través de la página de GitHub del proyecto.
