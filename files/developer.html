Cristopher De La O

Originalmente este proyecto no estaba planeado ya que utilizaba un hoja de Microsoft Excel para hacer el análisis y generar mis propios horarios, sin embargo la escalabilidad era muy baja y me encontré en la situación de tener que modificar formulas y mover datos.

Por el motivo anterior, decidí implementar una aplicación para llevar a cabo la tarea. Con el posible afín de ayudar a otros, implemente la aplicación en vanilla JavaScript para que cualquiera en la comunidad pueda utilizar la aplicación sin necesitad de instalaciones ni compilaciones de código.

Cualquier sugerencia para la mejora de esta aplicación es aceptada, puedes crear una petición en la página de GitHub
